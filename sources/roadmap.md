# BlackBox Project Roadmap

Last updated: 2026-04-22

## Purpose

This folder is the repo-maintained source of truth for:

- milestone numbering
- completed milestone archive
- current platform state
- next planned work
- documentation and Git hygiene expectations

## Strategic Phases

### Step 9 - Usability & Stabilisation

Priority:

1. `9.0` Project tracking and archive hygiene
2. `9.1` Canvas identity and collaboration clarification
3. `9.2` OpenWebUI result UX improvements
4. `9.3` Link reliability
5. `9.4` Model performance
6. `9.5` User provisioning
7. `9.6` OpenWebUI as the primary interface

Goal:

- make the current system usable, understandable, and stable

### Step 10 - AI In Canvas

Priority:

1. `10.1` Basic AI interaction in Canvas
2. `10.2` PhotoPrism search inside Canvas
3. `10.3` Structured Canvas object types
4. `10.4` Lightweight deterministic interaction model

Goal:

- turn Canvas into an AI-assisted thinking surface

### Step 11 - Embeddings & Semantic Search

Priority:

1. `11.1` Embedding pipeline
2. `11.2` Semantic search API
3. `11.3` OpenWebUI integration
4. `11.4` Canvas integration

Goal:

- upgrade image retrieval from keyword search to semantic search

## Working Rules

- Work on one milestone or sub-step at a time unless explicitly redirected.
- Update this roadmap when milestone numbering or sequencing changes.
- Keep Step 9, Step 10, and Step 11 clearly separated.
- Do not treat this document as an execution prompt; it is planning context.
